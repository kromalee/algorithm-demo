## 0. 学习算法和刷题的框架思维

- 数据结构的存储方式
  - 顺序存储:数组
  - 链式存储:链表

  - 高级结构:队列,栈,图,散列表,树 根据解决问题的不同都有数组和链表的分别实现

- 数据结构的基本操作
  - 线性遍历访问:数组遍历,链表遍历
  - 非线性遍历访问: 链表遍历,二叉树的遍历

- 刷题指南
  - 链表数组常用(小而美的)算法
  - 二叉树
  - 动态规划,回溯等...
